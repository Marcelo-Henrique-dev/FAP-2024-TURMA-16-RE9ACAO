# Plataforma de Integração Alunos × Universidades — Re9Ação

Projeto desenvolvido em equipe durante o programa **FAP Softex** 
(Formação Acelerada em Programação), para a empresa Re9Ação como 
cliente real, com foco em Aprendizagem Baseada em Projetos (PBL).

## Sobre o projeto
Plataforma web que conecta estudantes e universidades, facilitando 
o acesso a experiências práticas por meio de projetos reais.

## Tecnologias utilizadas
- ReactJS + TypeScript
- Tailwind CSS
- Zod (validação de formulários)
- Vite
- Metodologia Scrum / Trello
